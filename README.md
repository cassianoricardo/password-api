# Password API

API responsável por validar as senhas informadas.

Esta API REST desenvolvida utilizando:

    * Spring boot 2.6.7;
    * Swagger 3;
    * Lombok;
    * JUnit 5; 
    * MockMvc

## Pré requisitos

* JDK 11
* Apache Maven 3.8.5
